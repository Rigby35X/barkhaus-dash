üìã How to Generate Your Own Embed:

  1. Go to Dashboard:
    • Visit: http://localhost:4173/app/sites
    • Scroll down to "Animal Embed Widget"
  2. Configure Settings:
    • Set Base URL to: http://localhost:4173
    • Choose your preferred colors and filters
    • Set token expiration (1-24 hours)
  3. Generate & Test:
    • Click "Generate Embed Code"
    • Copy the HTML code
    • Paste it into any website
